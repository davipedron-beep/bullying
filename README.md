# Bullying Stop - Site sobre Bullying e Ciberbullying

Um site completo em **HTML5 puro** com CSS e JavaScript vanilla, dedicado à conscientização, prevenção e combate ao bullying e ciberbullying.

## 📋 Conteúdo

O site contém as seguintes páginas:

### 1. **index.html** - Página Inicial
- Hero section com chamada para ação
- Estatísticas sobre bullying
- Cards com links para principais seções
- Seção de depoimentos
- Call-to-action para ajuda imediata

### 2. **o-que-e-bullying.html** - O que é Bullying?
- Definição de bullying e ciberbullying
- Diferenças entre os dois
- Tipos de bullying (físico, verbal, social, psicológico, sexual, material)
- Características do bullying
- Mitos e verdades

### 3. **impactos.html** - Impactos e Consequências
- Impactos psicológicos para vítimas
- Impactos físicos
- Impactos acadêmicos
- Impactos sociais
- Sinais de alerta

### 4. **como-agir.html** - Como Agir
- Estratégias para vítimas de bullying
- Orientações para pais
- Orientações para testemunhas
- Passos práticos e acionáveis

### 5. **recursos.html** - Recursos e Apoio
- Canais de ajuda imediata (CVV, Disque Denúncia, etc.)
- Organizações e programas
- Materiais educativos para download
- Comunidades online

### 6. **faq.html** - Perguntas Frequentes
- Accordion interativo com 10 perguntas frequentes
- Respostas detalhadas
- Informações sobre responsabilidade legal

### 7. **contato.html** - Contato
- Formulário de contato
- Informações de contato
- Canais de emergência
- Links para ajuda imediata

## 🚀 Como Usar

### Opção 1: Abrir Localmente
1. Baixe todos os arquivos
2. Coloque-os em uma pasta
3. Abra `index.html` no navegador

### Opção 2: Usar em um Servidor Web
```bash
# Com Python 3
python -m http.server 8000

# Com Node.js (http-server)
npx http-server

# Com PHP
php -S localhost:8000
```

Acesse `http://localhost:8000` no navegador.

### Opção 3: Deploy em Plataforma
- **Netlify:** Drag & drop dos arquivos
- **GitHub Pages:** Faça upload para um repositório
- **Vercel:** Conecte seu repositório
- **Qualquer servidor web:** FTP/SFTP

## 🔧 Customização

### Mudar Cores
Edite as variáveis CSS em `styles.css`:
```css
:root {
    --primary: #5A7A8C;
    --secondary: #6DD5D5;
    --accent: #E8A8A0;
    /* ... */
}
```

### Adicionar Páginas
1. Copie um arquivo HTML existente
2. Modifique o conteúdo
3. Adicione o link no sidebar

### Mudar Fontes
Edite o import de Google Fonts em cada HTML:
```html
<link href="https://fonts.googleapis.com/css2?family=..." rel="stylesheet">
```
